	
# ts-p5-sor
 
Training for [Typescript](https://www.typescriptlang.org/) with [p5.js](https://p5js.org/) library
  
# Features
 
Making a canvas, then doing QuickSort with randomized lots of numbers.
 
# Requirement
 
* yarn
 
 
 # Command
```
yarn install
yarn watch
```
 
# Installation
 
Install Pyxel with pip command.
 
```bash
pip install pyxel
```
